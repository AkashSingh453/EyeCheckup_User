# EyeCheckup User

## Project Features

1. **Eye Power Checkups**: A comprehensive feature that allows users to check their eye power using various tests and questionnaires.
2. **Doctor Prescription**: Based on the results from the eye power checkup, users receive prescriptions from certified doctors specifically tailored to their needs and distance requirements.
3. **ML Model Integration**: Utilizes machine learning models to accurately detect the distance through camera analysis, enhancing the precision of the checks.
4. **Appointment Booking System**: Users can seamlessly book appointments with eye specialists, managing their schedules effectively to ensure timely consultations.

This project aims to provide users with a holistic approach to eye care, making the process efficient and user-friendly.
